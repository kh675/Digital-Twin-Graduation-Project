Error Messages:

=============================================================================
BATCH GENERATION
=============================================================================

Process:
1. Load all student data
2. Iterate through 1500 students
3. Generate charts for each student
4. Create PDF with embedded charts
5. Save to pdf_reports/ directory
6. Track progress with tqdm
7. Report success/failure summary

Performance:
- Estimated time: 5-10 minutes for 1500 students
- Memory usage: ~500MB
- Disk space: ~300-750 MB (1500 PDFs)
- Parallel processing: Can be added for speed

Output:
- 1500 PDF files in pdf_reports/
- Temporary charts in charts_temp/
- Naming: S0001_report.pdf to S1500_report.pdf
